# Keyboard shortcuts

Press **?** in the web UI to see the shortcuts overlay.

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+K` | Open command palette |
| `/` | Focus question input |
| `Cmd/Ctrl+B` | Toggle sidebar |
| `N` | New conversation |
| `D` | Toggle chat/dashboard view |
| `S` | Toggle chat/SQL editor view |
| `Cmd/Ctrl+Enter` | Run the query in the SQL editor |
| `Escape` | Close modal / blur input |
| `?` | Show shortcuts help |

On macOS, shortcuts use `Cmd`; on Windows/Linux, they use `Ctrl`.
